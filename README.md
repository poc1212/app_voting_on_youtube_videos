# About the project

The project was developed for a job application for full stack ReactJS and NodeJS developers.
The challenge was proposed by the contractor and I developed this app in 10 days.
The app was developed using ReactJS, NodeJS, MariaDB Database technologies

# Technologies Used
## Back end
- NodeJS
- Sequelize
- Sequelize-cli
- Mysql2
- Mariadb
- Dotenv
- Cors
- Bcryptjs
- Jsonwebtoken
## Front end
- HTML / CSS / JS 
- ReactJS
- React-router-dom
- Axios
- React-cookie
- React-icons
## Database
- MariaDB
# How to execute the project

## Back end
- npm install 
- npm init -y 
- cd node 
- node api.js 

## Front end
- npm install
- cd app
- npm run dev
